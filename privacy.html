개인정보처리방침

한눈체크는 생활 확인 도구 제공을 위해 사용자가 입력한 사업자 정보, 장소 검색어, 지역 선택값, 브라우저 기반 기기 확인값 등을 일시적으로 처리할 수 있습니다. 입력값은 해당 기능의 조회·확인 요청을 수행하기 위한 목적으로만 사용하며, 한눈체크 자체 데이터베이스에 원문 조회 이력을 별도 저장하지 않는 구조로 운영합니다.

처리할 수 있는 입력정보

  • 사업자등록번호, 대표자명, 개업일자
  • 상호명, 통신판매업 신고번호, 쇼핑몰 주소 또는 도메인
  • 장소 검색어, 지역 선택값, 지도 중심 좌표, 현재 위치 확인 시 브라우저가 제공하는 위치 좌표
  • 브라우저가 제공하는 기기·화면·성능 관련 값
  • 기능 오류 문의를 위해 사용자가 직접 보내는 문의 내용

위 정보는 사업자등록 상태조회, 사업자 진위확인, 통신판매업 신고정보 조회, 쇼핑몰 정보 비교, 전기차 충전소·주차비·미세먼지 확인 기능을 제공하기 위해 일시적으로 처리됩니다.

저장하지 않는 정보

  • 주민등록번호, 계좌번호, 전화번호, 개인 메신저 아이디를 조회 대상으로 삼지 않습니다.
  • 피해사례, 채팅 캡처, 송금 내역 등 민감한 거래 자료를 접수하거나 공개하지 않습니다.
  • 결과 공유 링크에 원문 사업자번호를 담지 않습니다.

위치 정보와 지도 기능

현재 위치 확인 기능을 사용할 때 브라우저가 제공하는 위치 좌표를 일시적으로 사용할 수 있습니다. 이 좌표는 주변 충전소, 주차장, 대기질 확인 기준점을 정하는 데 사용되며, 한눈체크 자체 데이터베이스에 위치 이력을 저장하지 않습니다.

장소 검색과 지도 표시 과정에서는 카카오 Local API 및 Kakao Maps JavaScript API가 사용될 수 있으며, 해당 서비스의 처리 과정에서 별도의 접속 로그가 생성될 수 있습니다.

외부 처리 및 로그

조회 요청은 Cloudflare Pages Functions를 통해 공식 API 제공기관으로 전달됩니다. 한눈체크가 조회값을 별도 DB에 저장하지 않더라도 Cloudflare, 공공데이터포털, API 제공기관의 시스템 처리 과정에서 접속 로그 또는 요청 기록이 생성될 수 있습니다.

광고와 쿠키 안내

향후 Google AdSense 등 광고가 적용될 수 있습니다. Google 및 제3자 광고 제공업체는 쿠키 또는 모바일 광고 식별자와 유사한 기술을 사용할 수 있습니다. Google의 광고 쿠키 사용을 통해 Google과 파트너는 사용자의 본 사이트 및 다른 사이트 방문 정보를 바탕으로 맞춤 광고를 게재할 수 있습니다.

사용자는 Google 광고 설정에서 맞춤 광고를 선택 해제할 수 있으며, 브라우저 설정을 통해 쿠키 저장을 제한할 수 있습니다. 광고 개인화 선택 해제, 쿠키 차단, 브라우저 추적 방지 설정은 사용자가 이용하는 브라우저와 Google 계정 설정에서 관리할 수 있습니다.

광고가 도입되더라도 조회 버튼, 전화 버튼, 지도 버튼과 혼동되지 않도록 광고 영역을 기능 UI와 분리해 표시합니다. 심사 전 배포본에서는 광고 스크립트를 비활성화합니다.

문의

개인정보 처리, 조회값 표시, 데이터 정정 요청과 관련한 문의는 문의 페이지를 통해 접수해 주세요.

기기·환경 체크 기능의 처리 방식

컴퓨터 사양 확인 기능에서 표시되는 운영체제 추정, CPU 논리 코어 수, 메모리 추정값, GPU/WebGL 정보, 화면 해상도, CPU 간단 테스트 결과는 사용자의 브라우저에서 즉시 확인·계산되는 정보입니다. 한눈체크는 해당 값을 자체 데이터베이스에 저장하지 않습니다.

정확한 CPU 모델명, RAM, GPU, 저장장치 등 사용자가 직접 입력한 값도 현재 화면의 해석 결과를 표시하기 위한 용도로만 사용되며, 서버에 별도 저장하지 않습니다.

주차비 확인 지도 안내

주차비 확인 지도에서 입력한 목적지와 위치 정보는 주차장 후보 확인을 위해 사용되며, 한눈체크 자체 DB에 저장하지 않는 것을 원칙으로 합니다.

한눈체크

생활 속에서 확인이 필요한 정보를 한눈에 정리하고, 결과의 의미와 한계를 함께 설명하는 생활 확인 도구입니다. 조회·측정 결과는 참고용이며, 실제 판단을 보장하지 않습니다.
