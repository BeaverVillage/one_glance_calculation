홈/컴퓨터 사양 확인

Device check

컴퓨터 사양을 한눈에 확인하세요

브라우저에서 확인 가능한 정보는 자동으로 정리하고, 정확한 CPU·GPU·RAM·저장장치 정보는 직접 입력해 용도별 의미를 해석할 수 있습니다.

내 컴퓨터 자동 확인

안내: 웹페이지는 보안 정책상 정확한 CPU 모델명, 전체 RAM 용량, 저장장치 모델명, 메인보드 정보를 직접 읽을 수 없습니다. 자동 확인값은 참고용이며, 부족한 항목은 직접 입력해서 해석합니다.

GPU 자동 확인값은 WebGL 렌더러일 수 있습니다. 실제 그래픽카드 모델명을 입력하면 해석이 더 정확합니다.

Manual input

정확한 부품명은 직접 입력하세요

자동 확인값만으로는 실제 CPU·GPU 모델을 확정하기 어렵습니다. 도움말 버튼으로 확인 위치를 보고 입력하거나, 사양표 이미지·PDF 또는 클립보드 붙여넣기 인식으로 후보값을 채울 수 있습니다.

사양표 전체가 보이는 캡처 이미지 또는 텍스트가 포함된 PDF를 선택하세요.

직접 입력한 사양, 사양표 이미지·PDF 인식 결과, 클립보드 붙여넣기 결과는 현재 브라우저 화면에서만 사용하며, 한눈체크 자체 DB에 저장하지 않습니다.

Meaning

사양 해석은 팝업에서 확인합니다

자동 확인 사양, 직접 입력 사양, CPU 간단 테스트 결과는 각각 버튼을 눌렀을 때 팝업으로 정리됩니다. 팝업에서는 확인값, 해석, 한계를 함께 보여줍니다.

Privacy

브라우저 안에서 처리합니다

자동 확인값, 직접 입력값, 캡처 이미지의 텍스트 인식 결과는 현재 페이지 안에서만 사용합니다. 사양 정보와 이미지를 한눈체크 서버 DB에 저장하지 않습니다.

사용 방법

자동 확인과 직접 입력을 나눠서 더 정확하게 봅니다.

  1. 자동 정보 확인
    운영체제 추정, 논리 코어 수, 메모리 추정, GPU/WebGL, 화면 해상도를 확인합니다.
  2. GPU·부품명 보완
    정확한 CPU 모델명, RAM, GPU, 저장장치는 직접 입력하거나 사양표 이미지·PDF, 클립보드 붙여넣기 인식으로 후보값을 채웁니다.
  3. CPU 간단 테스트
    짧은 반복 연산으로 현재 브라우저 환경의 참고 처리량을 확인합니다.
  4. 팝업 결과 확인
    각 결과 팝업에서 사양의 의미, 비교군, 한계를 함께 확인합니다.

한눈체크의 방향

정확히 알 수 있는 것과 알 수 없는 것을 구분합니다.

웹페이지는 보안상 모든 하드웨어 정보를 자동으로 읽을 수 없습니다. 그래서 한눈체크는 자동 확인값을 무리하게 단정하지 않고, 직접 입력과 도움말, 이미지 텍스트 인식으로 부족한 정보를 보완합니다.

CPU 이름 읽는 법

i3·i5·i7과 세대 숫자를 같이 봐야 합니다.

CPU 이름은 단순히 i7이면 무조건 좋고 i5면 무조건 낮다는 식으로 보면 안 됩니다. 같은 i5라도 최신 세대는 오래된 i7보다 체감이 좋을 수 있습니다.

  • i3 / Ryzen 3: 문서, 웹, 온라인 강의 중심의 보급형·기본 작업용으로 보는 경우가 많습니다.
  • i5 / Ryzen 5: 일반 사용, 개발 입문, FHD 게임 보조까지 많이 쓰이는 중급 라인입니다.
  • i7 / Ryzen 7: 멀티태스킹, 게임, 편집 작업에 더 여유가 있는 상급 라인입니다.
  • i9 / Ryzen 9: 고성능 데스크톱이나 상급 작업용 노트북에서 많이 보는 최상급 라인입니다.
  • 12·13·14처럼 앞 숫자: Intel i7-13620H에서 13은 13세대를 뜻합니다. Ryzen 7840HS에서 7은 7000번대 제품군을 보는 기준이 됩니다.
  • H·HX·U 같은 뒤 글자: H/HX는 고성능 노트북용, U는 저전력 노트북용, F는 내장그래픽이 빠진 데스크톱 CPU에서 자주 보입니다.

GPU 이름 읽는 법

RTX 4060의 40은 세대, 60은 등급을 보는 힌트입니다.

그래픽카드는 세대와 등급을 함께 봐야 합니다. RTX 4060에서 40은 40 시리즈 세대, 60은 같은 세대 안의 메인스트림 등급으로 이해하면 쉽습니다.

  • RTX 50·40·30 시리즈: 앞 두 자리는 세대를 보는 기준입니다. 50 시리즈가 더 최신이고, 40·30 시리즈 순으로 이전 세대입니다.
  • 50 / 60 / 70 / 80 / 90: 뒤 두 자리는 같은 세대 안의 등급을 보는 데 도움이 됩니다. 60은 메인스트림, 70은 중상급, 80·90은 상급·최상급에 가깝습니다.
  • Laptop 표기: 노트북 GPU는 같은 RTX 4060이라도 전력 제한, 냉각 설계, 전원 모드에 따라 체감 성능 차이가 큽니다.
  • Radeon RX 7600·7800: 첫 숫자는 세대, 그 다음 숫자는 같은 세대 안의 등급을 보는 데 도움이 됩니다.
  • 내장 GPU: Intel Iris Xe, Intel UHD, Radeon 780M 같은 내장 그래픽은 문서·영상 시청에는 충분할 수 있지만, 외장 GPU와는 용도가 다릅니다.

CPU 테스트 해석

간단 테스트 점수는 절대 순위가 아니라 현재 브라우저의 참고 처리량입니다.

CPU 간단 테스트는 현재 브라우저 안에서 짧은 반복 연산을 수행해 참고 점수를 보여줍니다. 전원 모드, 절전 설정, 발열, 백그라운드 작업, 브라우저 탭 수에 따라 결과가 달라질 수 있습니다.

  • 높은 점수: 현재 환경에서 짧은 반복 연산 처리량이 높은 편이라는 뜻입니다.
  • 낮은 점수: 실제 CPU가 낮은 성능일 수도 있지만, 절전 모드나 발열 제한 때문에 낮게 나올 수도 있습니다.
  • 비교군: 팝업의 비교군은 체감 이해를 돕기 위한 표현이며, 전문 벤치마크 순위를 대체하지 않습니다.

안전한 안내 기준

확인 결과는 참고용으로 설명하고, 성능을 보장하지 않습니다.

한눈체크는 입력한 사양을 바탕으로 일반적인 의미를 정리하지만, 특정 게임 실행 가능 여부나 전문 작업 성능을 보장하지 않습니다. 같은 부품명이라도 노트북 전력 제한, 발열, 드라이버, 메모리 구성, 저장장치 용량에 따라 체감은 달라집니다.

운영 원칙: 확인 가능한 값은 보여주고, 자동 확인이 제한되는 값은 직접 입력과 도움말로 보완합니다. 사양표 이미지·PDF와 붙여넣기 인식 결과는 현재 브라우저에서만 사용합니다.

FAQ

컴퓨터 사양 확인 자주 묻는 질문

왜 정확한 CPU 모델명이 자동으로 안 나오나요?

웹브라우저는 보안과 개인정보 보호 때문에 CPU 모델명, 저장장치 모델명, 메인보드 정보 같은 하드웨어 세부 정보에 직접 접근할 수 없습니다.

CPU 간단 테스트 결과는 실제 CPU 성능표와 같나요?

아닙니다. 브라우저 안에서 짧게 실행하는 참고 테스트입니다. 결과 팝업의 CPU 비교군은 절대 성능 순위가 아니라 체감 범위를 이해하기 위한 보조 설명입니다.

캡처 이미지 텍스트 인식은 정확한가요?

완벽하지 않습니다. OCR 결과는 자동 입력 후보이므로 조회 전 사용자가 반드시 모델명을 다시 확인해야 합니다.

입력한 사양이나 이미지가 저장되나요?

저장하지 않습니다. 직접 입력한 값과 이미지 텍스트 인식 결과는 현재 페이지에서 사양 해석을 보여주는 데만 사용됩니다.
