# 생활지도 15차 홈·메뉴 정리

## 목적

생활지도 3종을 만든 뒤 메인 화면과 주요 기능 사이드바가 과도하게 길어지는 문제를 정리했다.

## 반영 내용

- 홈 오른쪽 `자주 쓰는 기능`은 5개로 고정했다.
- `필요한 기능을 선택하세요` 영역에는 신규 3개 기능을 포함하되, 사용자 체감 가치 기준으로 공중화장실 → 무료 와이파이 → 낚시터 순으로 재정렬했다.
- 홈 JSON-LD ItemList 순서도 실제 화면 순서와 맞췄다.
- 주요 기능 드로어에 `생활 지도` 그룹을 추가하고 공중화장실, 무료 와이파이, 낚시터를 넣었다.
- `app.js` 캐시 버전을 `20260623-v122-home-menu-navigation`으로 갱신했다.

## 검증 기준

- 자주 쓰는 기능 카드가 정확히 5개여야 한다.
- 신규 3개 기능이 필요한 기능 영역에 있어야 한다.
- 주요 기능 드로어에 신규 3개 기능이 있어야 한다.
- 홈 JSON-LD 순서와 화면 순서가 일치해야 한다.
