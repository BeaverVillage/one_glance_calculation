# 생활지도 3종 3회 안정화 검증

## 목적

낚시터 찾기, 무료 와이파이 찾기, 공중화장실 찾기의 PC·모바일 런타임 계약을 배포 전 3회 반복 검증한다.

## 13차 보강

- 무료 와이파이와 공중화장실의 `loadDistrict()`에서 `index.json` 로딩 실패가 try/catch 밖으로 빠질 수 있던 구조를 보강했다.
- `requestId`를 `index.json` 요청 전에 먼저 발급해 빠른 지역 변경 시 이전 요청이 뒤늦게 화면을 덮지 않도록 정리했다.
- 캐시 로딩 실패 시 `currentDistrict`를 안전하게 비우고 빈 목록·fallback 안내를 렌더링하도록 했다.
- v119 버전 쿼리로 생활지도 3종 JS/CSS 캐시 무효화를 갱신했다.

## 검증 명령

```powershell
node scripts\verify-life-maps-triple-stability.js
```

이 검증은 다음을 포함한다.

1. 1차: HTML/JS/캐시 기본 계약 검증
2. 2차: PC 레이아웃·모바일 바텀시트·지구/시군구 로딩 안정성 검증
3. 3차: 전체 계약 재검증 및 배포 제외 파일·25MB 단일 파일 제한 검증
