# 주차비 계산 지도 MVP 메모

## 최종 MVP 방향

이번 MVP는 길안내 기능이 없는 카카오맵 기반 주차비 예산 계산 지도다.

핵심 범위:

- 카카오맵 JavaScript API로 지도와 마커 표시
- 카카오 Local API로 목적지 검색과 좌표 변환
- 공공데이터 기반 주차장 후보 조회 구조
- 목적지 좌표와 주차장 좌표 사이의 직선거리 계산
- 입차/출차 시간, 빠른 시간 선택, 할인 조건 기준 예상 주차비 계산
- 일주차 전환점, 실시간 빈자리, 만차 위험도, 데이터 신뢰도 표시
- 모바일 바텀시트 형태의 추천 결과 UI

## 이번 MVP에서 제외한 항목

- 출발지 입력
- 자동차 경로 계산
- 차량 이동시간 계산
- 차량 이동거리 계산
- 유류비/통행료 계산
- 외부 경로 API 연동
- 차량 시간 기준 정렬
- 네이버 지도/네이버 Directions/구글 지도 연동

제외 이유: 경로 API는 비용, 약관, 호출량 제한 리스크가 있으므로 이번 MVP는 주차비 계산과 지도 표시 자체에 집중한다.

## 필요한 환경변수

- `KAKAO_MAP_JS_KEY`: 지도 표시용 JavaScript Key
- `KAKAO_REST_API_KEY`: 카카오 Local API 목적지 검색용 REST Key. Functions에서만 사용
- `PUBLIC_DATA_API_KEY`: 공공데이터포털 주차장/특일 정보 연동용
- `SEOUL_OPEN_API_KEY`: 서울 열린데이터 주차장/실시간 빈자리 연동용
- `HOLIDAY_API_KEY`: 한국천문연구원 특일 정보 공휴일 판정용

## API 라우트

- `GET /api/parking/places?q=검색어`
- `GET /api/parking/lots?lat=...&lng=...&radius=...`
- `POST /api/parking/estimate`
- `GET /api/parking/realtime?parkingLotId=...`
- `POST /api/parking/recommend`
- `GET /api/parking/holidays?date=YYYY-MM-DD`

## 추천 점수

추천점수는 다음 요소만 사용한다.

- 요금점수: 할인 후 주차비가 낮을수록 높음
- 거리점수: 목적지와 직선거리가 가까울수록 높음
- 만차위험점수: 위험도가 낮을수록 높음
- 데이터신뢰도점수: 신뢰도가 높을수록 높음
- 공영가산점: 공영 후보에 소폭 가산

정렬 옵션:

- 추천순
- 저렴한순
- 가까운순
- 빈자리순
- 신뢰도순

## 데이터 표시 원칙

- 실시간 가능 대수는 제공기관 데이터 기준이며 현장 상황과 다를 수 있다.
- 할인 계산은 선택한 할인율을 적용한 참고 계산이다.
- 목적지와 주차장 사이의 거리는 좌표 기반 직선거리다.
- 실제 요금, 운영시간, 할인 조건, 주차 가능 여부는 현장 또는 관리기관 안내를 확인해야 한다.
