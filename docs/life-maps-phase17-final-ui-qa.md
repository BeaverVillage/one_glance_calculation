# 생활지도 3종 17차 최종 UI QA

## 목적
낚시터 찾기, 무료 와이파이 찾기, 공중화장실 찾기 3개 기능을 PC와 모바일 기준으로 다시 점검하고, 배포 전 UI 이상 가능성을 줄인다.

## 주요 보강
- 생활지도 3종 CSS/JS 버전 쿼리를 `20260623-v126-life-maps-ui-polish-final`로 갱신했다.
- PC 레이아웃에서 왼쪽 목록 폭을 제한하고 지도 영역을 우선 확장하도록 보강했다.
- 목록 카드에는 `content-visibility`와 안전 줄바꿈을 적용해 카드 겹침 가능성을 줄였다.
- 모바일 목록 바텀시트 결과 영역에 내부 스크롤, 드래그 터치 안정화, safe-area 여백을 보강했다.
- 모바일 필터 액션시트의 스크롤과 체크박스 터치 영역을 보강했다.
- 무료 와이파이 UI에서 `와이파이 이름 있는 곳만` 대신 `와이파이 이름 있는 곳만`으로 사용자 중심 문구를 적용했다.
- 낚시터 상태 문구에서 거리 기준을 `현재 위치 기준`, `서울 중심 기준`처럼 명확히 표시하도록 정리했다.

## 검증
`node scripts/verify-life-maps-final-ui-qa.js`로 다음 항목을 확인한다.

- 생활지도 3종 HTML/JS 존재
- v124 버전 쿼리 적용
- 목록 클릭, 상세 카드 닫기, 카카오맵 바로가기 문구 존재
- 모바일 바텀시트와 필터 액션시트 구조 존재
- 추천순, 거리 기준 라벨, 마커 제한, requestId 레이스 방지 존재
- 무료 와이파이 문구가 사용자 중심으로 변경됐는지 확인
- 전체 도구 페이지의 app.js 버전 쿼리 일관성 확인
