# 생활지도 14차 작업 기록

## 목적
낚시터 찾기, 무료 와이파이 찾기, 공중화장실 찾기 3개 기능의 정렬 기준과 모바일 필터 UX를 기존 지도형 기능 수준에 맞게 보강했습니다.

## 주요 변경
- 기본 정렬을 가까운 순에서 추천순으로 변경했습니다.
- 거리만 보지 않고 실제 사용에 유용한 정보 보유 여부를 점수에 반영했습니다.
- 모바일에서 상세 필터 버튼을 누르면 하단 액션시트가 열리도록 했습니다.
- 액션시트에는 닫기, 초기화, 기존 적용 버튼이 포함됩니다.
- 모바일 목록 바텀시트에 드래그 닫기/확장 동작을 추가했습니다.

## 추천순 기준
### 낚시터
거리, 전화번호, 요금 정보, 주요어종, 안전시설, 편의시설, 실내낚시터 여부를 반영합니다.

### 무료 와이파이
거리, 와이파이 이름 제공 여부, 장소 상세, 시설구분, 제공기관, 관리 전화 여부를 반영합니다.

### 공중화장실
거리, 상시개방, 장애인 화장실, 기저귀교환대, 비상벨, CCTV, 관리 전화 여부를 반영합니다.

- PC 인터랙션 검증 스크립트
