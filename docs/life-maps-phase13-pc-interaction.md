# 생활지도 13차 PC UI·기본 인터랙션 안정화

## 목적

낚시터 찾기, 무료 와이파이 찾기, 공중화장실 찾기 3개 지도형 기능을 기존 주차비·응급실·전기차 충전소 지도와 더 가깝게 맞추기 위한 PC 우선 안정화 작업이다.

## 반영 내용

- PC 레이아웃을 `왼쪽 목록 340~410px + 오른쪽 지도 확장` 구조로 보정
- 지도 최소 높이를 640px 이상으로 키워 기존 지도형 기능과 시각적 밀도 맞춤
- 목록 카드 전체 클릭 시 지도 선택 카드가 뜨도록 구현
- Enter/Space 키보드 선택 지원
- 선택 카드 오른쪽 상단 X 닫기 버튼 추가
- 하단 닫기 버튼과 상단 X 버튼 모두 동작하도록 연결
- 거리 표시 기준을 현재 위치, 검색 위치, 지역/시군구 중심 기준으로 표시
- `지도 확인` 문구를 `카카오맵 바로가기` 또는 `카카오맵 검색`으로 변경
- 낚시터는 장소명 검색이 중요하므로 `카카오맵 검색` 버튼 추가
- 무료 와이파이는 `와이파이 이름` 표기를 `와이파이 이름`로 바꾸고 `비밀번호: 현장 확인 필요` 정보를 명시
- 목록 카드 정보 박스를 compact하게 줄이고 지도 상세 카드에는 높이 제한과 내부 스크롤을 적용
