# 16차 작업: 생활지도·응급실 콘텐츠 보강

## 목적

신규 생활지도 3종과 응급실 확인 페이지가 단순 지도 도구처럼 보이지 않도록, 사용 방법·데이터 기준·결과 해석·방문 전 확인사항·FAQ·관련 기능 링크를 보강했다.

## 보강 페이지

- 낚시터 찾기
- 무료 와이파이 찾기
- 공중화장실 찾기
- 응급실·야간 병원 확인

## 주요 보강 내용

- 공공데이터 기반 참고 정보임을 명확히 설명
- 실제 현장 상황과 차이가 날 수 있는 항목 안내
- 거리 기준, 비밀번호, 상시개방, 중증 항목 등 사용자가 오해하기 쉬운 표현 설명
- 관련 기능 내부 링크 추가
- FAQPage JSON-LD 항목 확장
