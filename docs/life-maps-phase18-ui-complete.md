# 생활지도 3종 UI 완성 안정화

낚시터 찾기, 무료 와이파이 찾기, 공중화장실 찾기에서 PC 목록 카드, 지도 상세 카드, 모바일 바텀시트, 필터 액션시트, 카카오맵 버튼 문구, 무료 와이파이 용어, footer 표준화를 함께 안정화했다.

## 핵심 변경

- 왼쪽 목록 카드를 compact 형태로 축소하고 상세 정보 박스를 목록에서는 숨김 처리했다.
- 지도 위 상세 카드는 최대 폭과 높이를 제한해 지도를 과도하게 가리지 않도록 했다.
- `카카오맵 검색`과 `카카오맵 바로가기`가 함께 나오던 구조를 `카카오맵 바로가기` 하나로 통일했다.
- `와이파이 이름 있음`, `와이파이 이름` 등 기술 용어 중심 문구를 사용자 언어인 `와이파이 이름`, `비밀번호 현장 확인 필요`로 정리했다.
- 모바일 목록 바텀시트는 transform 기반으로 열림/확장/닫힘 상태를 처리하고, 드래그 중 순간이동을 줄였다.
- 생활지도 마커와 fallback 마커의 검정 그림자 효과를 제거했다.
- 전체 HTML footer를 메인 페이지와 같은 `container footer-grid` 구조로 통일했다.
