수수료 미포함

실제 은행, 카드사, 송금업체, 환전소의 적용 환율과 수수료는 다를 수 있습니다. 중요한 송금·환전·결제 전에는 이용하려는 기관의 고시 환율과 수수료를 확인하세요.

환율 계산 가이드

실시간 환율 계산기를 이렇게 활용하세요.

최신 환율, 직접 입력 환율, 최근 조회 환율을 상황에 맞게 선택하고 수수료와 우대율을 함께 확인할 수 있습니다. 결과는 실제 은행·카드사·송금업체 적용 금액이 아니라 참고용 예상값입니다.

이 계산기는 무엇을 도와주나요?

미국 달러, 일본 엔화, 유로, 호주 달러 등 여러 통화의 금액을 다른 통화로 환산하고, 예상 수령액과 반대 환율을 한 화면에서 비교할 수 있게 돕습니다.

환율 계산 기준

오늘 환율은 공개 환율 API에서 조회한 최신 시세를 사용합니다. 특정 은행 고시 환율이나 송금앱 환율이 있다면 수동 입력 모드로 직접 입력해 비교할 수 있습니다.

수수료와 우대율 반영 방식

입력한 수수료율과 우대율을 기준으로 실제 적용 수수료율을 계산하고, 고정 수수료가 있으면 결과 통화 기준으로 차감합니다. 은행별 수수료 실시간 조회 기능은 제공하지 않습니다.

실제 환전 금액과 차이가 나는 이유

환율은 조회 시점, 현찰 살 때·팔 때 기준, 카드 해외결제 수수료, 송금 수수료, 각 기관의 스프레드에 따라 달라질 수 있습니다. 실제 거래 전에는 이용 기관의 고시 환율을 확인해야 합니다.

자주 쓰는 통화쌍

USD/KRW, JPY/KRW, AUD/KRW, EUR/KRW, KRW/JPY, KRW/USD 빠른 선택 버튼으로 여행비, 해외급여, 직구비, 송금 예상액을 빠르게 가늠할 수 있습니다.

사용 시 주의사항

API가 일시적으로 응답하지 않거나 특정 통화를 지원하지 않을 수 있습니다. 이 경우 최근 조회 환율 또는 직접 입력 환율을 사용하고, 중요한 금융 거래 전에는 공식 고시 환율과 수수료를 확인하세요.

계산 기준

실시간 환율 계산 기준

환율, 수수료, 우대율을 함께 반영해 원화 환산액 또는 외화 환산액을 계산합니다.

이 계산기는 무엇을 계산하나요?

선택한 통화와 입력 금액을 기준으로 환율 적용 금액, 수수료 반영 금액, 우대율 적용 후 예상 금액을 계산합니다.

입력값 설명

  • 금액: 환산하려는 원화 또는 외화 금액입니다.
  • 통화: 환율을 적용할 기준 통화입니다.
  • 수수료와 우대율: 실제 환전·송금 조건을 참고해 직접 조정할 수 있습니다.

계산식

기본 환산액에 환율을 적용한 뒤, 입력한 수수료율과 우대율을 반영해 예상 부담액을 표시합니다.

예시 계산

100달러를 원화로 계산할 때 조회 환율이 1,350원이고 수수료 조건을 입력하면, 135,000원을 기준으로 추가 비용 또는 우대 적용 결과를 확인합니다.

주의사항

실제 적용 환율은 은행, 카드사, 송금 서비스, 거래 시점, 고시 기준에 따라 달라질 수 있습니다.

참고 안내

환율 조회가 실패하면 로컬 기준값 또는 마지막 조회값을 사용할 수 있으므로 실제 거래 전에는 이용하려는 금융기관의 최종 환율을 확인하세요.

FAQ

자주 묻는 질문

실시간 환율 계산기는 실제 은행 환율과 같은가요?

아니요. 공개 환율 API의 최신 시세를 참고용으로 보여주며, 은행, 카드사, 송금업체, 환전소의 실제 적용 환율과 수수료는 다를 수 있습니다.

환전 수수료는 어떻게 계산하나요?

기준 환산 금액에서 입력한 수수료율과 우대율을 반영한 예상 수수료를 차감하고, 고정 수수료가 있으면 결과 통화 기준으로 추가 차감합니다.

우대율은 무엇인가요?

우대율은 입력한 기본 수수료율 중 할인되는 비율입니다. 수수료 2%, 우대율 50%라면 계산기에서는 실제 적용 수수료율을 1%로 계산합니다.

송금 수수료와 카드 해외결제 수수료도 반영되나요?

별도 항목으로 자동 계산하지는 않습니다. 고정 수수료나 수수료율 입력칸에 직접 넣어 참고용으로 계산할 수 있습니다.

오늘 환율과 직접 입력 환율 중 무엇을 써야 하나요?

일반적인 확인에는 오늘 환율을 사용할 수 있고, 은행 고시 환율이나 송금앱에서 확인한 특정 환율이 있다면 직접 입력 모드가 더 적절합니다.

호주 달러를 원화로 바꿀 수 있나요?

네. AUD/KRW 빠른 선택을 누르거나 From에 AUD, To에 KRW를 선택하면 호주 달러를 원화로 환산할 수 있습니다.

원화를 엔화나 달러로 바꿀 수 있나요?

네. KRW/JPY, KRW/USD 빠른 선택을 사용하거나 직접 통화를 선택하면 원화 기준 환산도 가능합니다.
