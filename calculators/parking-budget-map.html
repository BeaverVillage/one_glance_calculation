이 계산기는 무엇을 계산하나요?

목적지 주변 주차장의 예상 요금, 내 출발지 기준 차량 이동시간, 실시간 빈자리 여부, 만차 위험도와 데이터 신뢰도를 함께 비교합니다.

주차비 계산 기준

기본요금과 추가요금을 기준으로 선택한 주차시간의 예상 금액을 계산하고, 일주차권이 더 유리한 시점이 있으면 함께 표시합니다.

일주차 전환점

시간요금 누적액이 일주차권 요금 이상이 되는 최소 시간을 계산합니다. 일주차 정보가 없으면 전환점은 표시하지 않습니다.

공영·무료·실시간 필터

공영주차장 우선, 무료 주차 가능성, 일주차 가능 여부, 실시간 빈자리 제공 여부를 필터로 좁혀 볼 수 있습니다.

할인 계산 방식

경차, 장애인, 전기차 또는 직접 입력 할인율을 적용한 참고 금액을 표시합니다. 실제 할인은 현장 정책과 증빙 확인에 따라 달라질 수 있습니다.

데이터 신뢰도

요금, 운영시간, 좌표, 기준일, 실시간 정보가 충분할수록 신뢰도를 높게 표시합니다. 정보가 일부 누락된 주차장은 현장 확인이 필요합니다.

사용 시 주의사항

본 결과는 공공데이터 및 실시간 제공 데이터를 바탕으로 계산한 예상 주차비입니다. 실제 요금, 할인 적용 여부, 운영시간, 주차 가능 여부는 현장 사정에 따라 달라질 수 있습니다. 방문 전 주차장 또는 관리기관에 확인해 주세요.

실시간 가능 대수는 제공기관 데이터 기준이며 실제 현장 상황과 차이가 있을 수 있습니다. 경차, 장애인, 전기차 할인은 주차장별 정책에 따라 다를 수 있으며, 선택한 할인율은 참고 계산용입니다.

주차비 예산 지도 FAQ

주차비 예산 지도는 실제 주차요금을 보장하나요?

아니요. 공공데이터와 입력값을 바탕으로 한 참고용 예상 금액입니다. 실제 요금, 할인, 운영시간은 현장 정책에 따라 달라질 수 있습니다.

실시간 빈자리가 있으면 주차 가능이 보장되나요?

보장하지 않습니다. 실시간 가능 대수는 제공기관 데이터 기준이며 현장 진입 시점의 실제 상황과 차이가 날 수 있습니다.

경차나 전기차 할인은 자동으로 확정 적용되나요?

아니요. 선택한 할인율을 적용한 참고 계산이며 실제 할인 적용 여부는 주차장별 정책, 증빙자료, 결제 방식에 따라 다를 수 있습니다.

API 키가 없어도 페이지를 테스트할 수 있나요?

예. 개발과 화면 확인을 위해 샘플 주차장 데이터와 거리 기반 fallback 계산을 제공합니다. 실제 운영 데이터는 Cloudflare Pages 환경변수와 API 연동 후 확인해야 합니다.
