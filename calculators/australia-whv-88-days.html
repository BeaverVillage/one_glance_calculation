호주 워홀 세컨비자 88일 근무일 계산기

급여명세서 PDF 또는 이미지 파일에서 Pay Period와 근무시간을 읽고, 세컨비자 88일 또는 서드비자 179일 기준의 specified work 인정 가능 일수를 참고용으로 계산합니다.

이 계산기는 법률·이민 자문이 아니라 참고용 도구입니다. 실제 second/third visa 신청 가능 여부를 보장하지 않으며, 최종 판단은 반드시 Australian Department of Home Affairs 공식 안내와 ImmiAccount 신청 기준을 확인해야 합니다.
  1. 1. 목표 비자와 subclass를 선택하세요.Second visa 88일 또는 Third visa 179일 기준을 고릅니다.
  2. 2. 계산 방식을 선택하세요.Pay Period가 있으면 자동 계산, 총 근무시간만 있으면 Hours 추정, 이미 일수를 알고 있으면 직접 입력을 사용합니다.
  3. 3. payslip PDF/이미지를 업로드하거나 값을 입력하세요.PDF와 이미지는 서버에 저장하지 않고 브라우저에서 처리됩니다.
  4. 4. 자동 추출값과 지역·업종 기준을 확인하세요.확인되지 않은 항목은 최종 인정 가능 일수에 포함하지 않습니다.
  5. 5. 88일 또는 179일 기준 달성률을 확인하세요.결과는 공식 확인 전 참고용 추정값입니다.

Step 3

자료 입력

선택한 계산 방식에 맞는 입력 영역만 표시됩니다. Pay Period가 있는 payslip은 자동 계산 모드를 가장 먼저 사용해 보세요.

급여명세서 PDF/이미지로 자동 계산

PDF, JPG, PNG, WEBP 파일을 지원합니다. PDF와 이미지는 서버에 저장하지 않고 브라우저에서 처리됩니다.

개인정보 보호 안내

이름, 주소, 직원번호, TFN, 은행 정보 등 민감정보는 가능한 한 가린 뒤 사용하세요. 사진 품질, 기울기, 그림자, 해상도에 따라 OCR 결과가 틀릴 수 있습니다. 자동 추출값은 반드시 직접 확인한 뒤 계산하세요.

사진은 글자가 선명하고 그림자가 적으며, 명세서 전체가 잘리지 않게 촬영한 파일을 사용하면 인식률이 높아집니다. 지원하지 않는 HEIC, TIFF, Word, Excel 파일은 처리하지 않습니다.

파일을 선택하면 PDF 분석 중 또는 이미지 분석 중 상태가 표시됩니다.

자동 계산 기준

Pay Period start와 end가 모두 있으면 두 날짜를 포함해 calendar days를 계산합니다. 예: 2026-10-05부터 2026-10-11까지는 7일입니다.

Pay Period가 없고 Hours Worked만 추출된 경우에만 각 카드에서 Hours Worked 추정 모드로 계산하는 버튼을 표시합니다.

payslip에 실제 근무일수나 Pay Period가 없고 Hours Worked만 있을 때 사용하는 보조 모드입니다.

확인 체크

추정 계산식

추정 근무일수 = Hours Worked ÷ 하루 기준 근무시간입니다. 예: 38시간 ÷ 7.6시간 = 약 5일입니다. 이 값은 실제 인정 일수가 아니라 근무시간을 기준으로 한 참고용 추정값입니다.

인정 일수 직접 입력

이미 payslip별 인정 일수를 직접 계산했거나 별도 증빙 기준으로 합산하고 싶을 때 사용합니다. 직접 입력값은 최종 합계에서 가장 우선 적용됩니다.

확인 체크

직접 입력 우선 적용

payslip 카드에서 자동 계산값과 직접 입력값이 다르면 사용자 직접 입력 일수를 최종 합계에 우선 적용합니다. 입력 근거와 메모를 남겨 두면 나중에 검토하기 쉽습니다.

각 항목의 자동 추출값을 수정하고, 업종·지역·값 확인 체크를 완료해야 최종 인정 가능 일수에 포함됩니다.

아직 입력된 payslip 또는 항목이 없습니다.

이 계산기는 무엇을 계산하나요?

호주 워킹홀리데이 second 또는 third visa 준비자가 payslip의 Pay Period, Hours Worked, 고용주 정보, 업종과 지역 확인 상태를 바탕으로 specified work 인정 가능 일수를 참고용으로 계산합니다.

세컨비자 88일 기준이란?

Second visa 준비 시 일반적으로 specified work 88일 기준을 확인해야 하는 경우가 있습니다. 이 계산기는 입력한 자료 기준으로 목표까지 남은 일수와 달성률을 보여줍니다.

서드비자 179일 기준이란?

Third visa는 더 긴 specified work 기간이 요구될 수 있어 179일 기준으로 합계를 볼 수 있습니다. 목표 비자를 Third visa로 바꾸면 결과 기준도 함께 바뀝니다.

지급명세서에서 어떤 정보를 읽나요?

Pay Period start/end, Pay Date, Pay Cycle, Hours Worked, Employer Name, ABN, Position, Employee Name을 우선 추출합니다. set2 테스트 자료처럼 Postcode, State, Industry가 문서에 있으면 함께 읽도록 parser를 사용합니다.

PDF와 이미지 파일은 어떻게 처리하나요?

PDF는 PDF.js로 텍스트 추출을 먼저 시도하고 부족하면 페이지를 canvas로 렌더링한 뒤 Tesseract.js OCR을 실행합니다. JPG, PNG, WEBP 이미지는 브라우저 canvas에서 리사이즈와 전처리를 거쳐 OCR합니다.

인정 일수 계산 방식

Pay Period가 있으면 시작일과 종료일을 포함한 calendar days로 계산합니다. Pay Period가 없고 Hours Worked만 있을 때는 하루 기준 근무시간으로 나눈 값을 내림해 추정으로 표시합니다. 직접 입력 일수가 있으면 해당 값을 우선 적용합니다.

지역과 업종을 사용자가 직접 확인해야 하는 이유

Home Affairs는 eligible postcode와 specified work 기준을 변경할 수 있습니다. 이 페이지의 postcode 확인은 local JSON 구조를 통한 참고 메시지이며, 현재 확정 range를 제공하지 않으므로 공식 기준 직접 확인이 필요합니다.

사용 시 주의사항

이 계산기는 실제 비자 신청 가능 여부를 보장하지 않습니다. 인정 업종, 인정 지역, 고용 형태, 근무 증빙, 여권 국가, 비자 subclass에 따라 결과가 달라질 수 있으며 필요한 경우 등록된 migration agent 또는 관련 전문가에게 확인하세요.

호주 워홀 88일 계산기 FAQ

급여명세서만 있으면 세컨비자 88일이 확정되나요?

아닙니다. 이 계산기는 Pay Period와 근무시간을 바탕으로 specified work 인정 가능 일수를 참고용으로 추정합니다. 실제 인정 여부는 업종, 지역, 고용 형태, 증빙자료, 비자 subclass와 공식 기준에 따라 달라질 수 있습니다.

Pay Period 전체 날짜가 모두 인정되나요?

Pay Period 전체가 specified work 조건을 충족한다고 사용자가 확인한 경우에만 calendar day 방식으로 포함합니다. 실제 인정 방식은 공식 Home Affairs 안내를 확인해야 합니다.

주급 payslip 하나는 무조건 7일로 인정되나요?

무조건은 아닙니다. 해당 기간의 업무와 근무지가 적격 조건에 해당하고 증빙이 충분한지 사용자가 직접 확인해야 합니다.

근무시간만 있을 때는 어떻게 계산하나요?

Hours Worked를 하루 기준 근무시간으로 나누고 소수점은 내림 처리해 참고용 추정 일수를 표시합니다. 기본 하루 기준은 7.6시간이며 사용자가 수정할 수 있습니다.

이미지 파일도 인식할 수 있나요?

JPG, PNG, WEBP 파일을 브라우저에서 canvas로 처리한 뒤 Tesseract.js OCR로 읽습니다. 사진 선명도, 기울기, 그림자, 해상도에 따라 결과가 달라질 수 있습니다.

영국 여권 소지자도 88일 계산이 필요한가요?

UK passport holder로 subclass 417 second 또는 third visa를 신청하는 경우 2024년 7월 1일 이후 specified work requirement가 면제될 수 있습니다. 공식 안내를 직접 확인하세요.

지역 postcode는 자동으로 판단하나요?

외부 API는 사용하지 않고 local JSON 구조만 제공합니다. 현재 버전은 확정 postcode range를 넣지 않아 대부분 공식 확인 필요로 표시됩니다. 틀린 자동 판정보다 직접 확인 안내를 우선합니다.

이 계산 결과를 비자 신청서에 그대로 제출해도 되나요?

아닙니다. 결과는 참고용이며 비자 승인 가능성을 보장하지 않습니다. 최종 판단은 Australian Department of Home Affairs와 ImmiAccount 신청 기준, 필요한 경우 전문가 확인을 따르세요.
