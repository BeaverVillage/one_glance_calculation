버튼 구성

일반 계산과 복소수 계산 모드를 분리하고, 기본·함수·상수·복소수 탭으로 버튼을 나눴습니다. 함수 탭에는 하이퍼볼릭 함수, 순열·조합, mod, 표시 모드 버튼을 넣었고, 복소수 탭에는 j, ∠, Abs, Arg, Conj, Re, Im, Pol, Rec 버튼을 넣었습니다. 모바일에서도 4열 고정 키패드를 사용합니다.

최근 계산

    사용 방법

    계산하고 싶은 식을 입력하세요.

    1. 식 입력키보드로 입력하거나 계산기 버튼을 눌러 식을 만듭니다.
    2. 각도 선택삼각함수를 쓸 때 문제에서 DEG, RAD, GRAD 중 어떤 단위를 쓰는지 확인합니다.
    3. 결과 확인계산 결과와 최근에 계산한 식을 함께 확인합니다.
    4. 다시 계산괄호나 숫자를 바꿔 다른 경우도 바로 비교해 봅니다.

    계산할 때 알아둘 점

    기호를 이렇게 이해합니다.

    • ^는 제곱이나 거듭제곱을 뜻합니다.
    • log()는 상용로그, ln()은 자연로그입니다.
    • nPr(n,r), nCr(n,r), mod(a,b)처럼 쉼표로 값을 나누는 함수도 사용할 수 있습니다.
    • mu0, eps0, c, R 같은 상수는 상수 탭에서 입력할 수 있습니다.
    • 복소수 모드에서는 3+4j, Abs(3+4j), Arg(3+4j), Conj(3+4j), 5∠30 같은 입력을 사용할 수 있습니다.
    • sqrt()는 제곱근입니다. 예를 들어 sqrt(16)은 4입니다.
    • 삼각함수는 선택한 각도 모드에 맞춰 계산됩니다.

    관련 팁

    실수를 줄이는 입력 팁

    • 괄호를 먼저 확인2+3*4와 (2+3)*4는 결과가 다릅니다.
    • 각도 단위 확인문제에서 도인지 라디안인지 먼저 확인한 뒤 계산하세요.
    • 너무 긴 식은 나누기중간값을 먼저 계산하면 입력 오류를 찾기 쉽습니다.

    계산 기준

    계산 전에 확인할 내용

    아래 내용은 계산 결과를 해석할 때 필요한 기준과 주의사항입니다.

    이 계산기는 무엇을 계산하나요?

    사칙연산, 괄호, 거듭제곱, 제곱근, 로그, 삼각함수, 하이퍼볼릭 함수, 순열·조합, 나머지 계산, 물리·전자기 공학 상수를 입력한 순서와 연산 우선순위에 맞춰 계산합니다.

    이런 상황에서 사용하면 좋습니다

    과제, 실험값 정리, 단위 변환 전 계산처럼 괄호와 함수가 섞인 계산을 빠르게 확인하고, DEG/RAD/GRAD 모드와 Norm/Sci/Eng 표시 차이를 점검할 때 사용하면 좋습니다.

    입력값 설명

    • 계산식: 숫자, 연산자, 괄호, 함수명을 입력합니다.
    • 각도 모드: 삼각함수를 도(DEG), 라디안(RAD), 그라디안(GRAD) 기준으로 계산합니다.
    • 표시 모드: 결과를 일반 표기, 과학 표기, 공학 표기 중 선택해 확인합니다.

    계산식

    입력식을 토큰으로 나눈 뒤 괄호, 함수, 팩토리얼, 거듭제곱, 곱셈·나눗셈, 덧셈·뺄셈 순서로 계산합니다. nCr(10,3), mod(17,5)처럼 2개 인수를 쓰는 함수는 쉼표로 구분합니다.

    예시 계산

    DEG 모드에서 sin(30)+sqrt(16)을 입력하면 0.5+4로 계산해 4.5를 표시합니다. nCr(10,3)은 120, mod(17,5)는 2로 계산합니다.

    결과 해석

    결과가 예상과 다르면 괄호 위치와 DEG/RAD 모드를 먼저 확인하세요.

    주의사항

    보고서, 시험, 실무에 사용할 때는 유효숫자, 반올림, 단위 기준을 별도로 확인해야 합니다.

    참고한 공식 자료 또는 안내 문구

    수학 표기와 브라우저 JavaScript Math 함수 기준을 사용합니다. 공학 상수는 CODATA 계열의 널리 쓰이는 고정값 또는 표준값을 참고 계산용으로 사용합니다.

    자주 묻는 질문

    공학용 계산기 자주 묻는 질문

    sin(30)은 0.5로 계산되나요?

    각도 모드가 DEG로 선택되어 있으면 sin(30)은 30도로 계산되어 0.5가 됩니다. RAD에서는 30라디안, GRAD에서는 30그라디안으로 계산됩니다.

    어떤 계산까지 할 수 있나요?

    현재는 sin, cos, tan, asin, acos, atan, sinh, cosh, tanh, asinh, acosh, atanh, log, ln, sqrt, abs, exp, x!, nPr, nCr, mod, floor, ceil, round, 공학 상수, 복소수 j 계산, Abs·Arg·Conj·Re·Im, Pol·Rec, 페이저 입력을 사용할 수 있습니다.

    계산 기록은 저장되나요?

    최근 계산 기록은 화면에서만 잠깐 보여줍니다. 새로고침하면 사라집니다.

    보고서나 과제에 그대로 써도 되나요?

    계산 확인용으로 쓰기 좋습니다. 보고서나 과제에 넣을 때는 유효숫자, 반올림 기준, 단위를 한 번 더 확인하세요.
