공학용 계산기

과제나 실험값을 정리하다가 계산이 필요할 때 바로 사용하세요. 사칙연산, 제곱근, 로그, 삼각함수까지 한 번에 계산할 수 있습니다.

sin(30)+√(16)
Ans -

키보드로 입력하거나 아래 버튼을 눌러 계산식을 만들 수 있습니다.

각도 모드

이렇게 입력해 보세요

예: 2^10, sqrt(2), log(100), sin(30), 3*(4+5)

버튼 구성

위쪽은 삼각함수와 로그, 가운데는 괄호와 제곱, 아래쪽은 숫자와 사칙연산입니다. 키보드로 직접 입력해도 화면에 바로 반영됩니다.

최근 계산

    사용 방법

    계산하고 싶은 식을 입력하세요.

    1. 식 입력키보드로 입력하거나 계산기 버튼을 눌러 식을 만듭니다.
    2. 각도 선택삼각함수를 쓸 때 문제에서 도를 쓰는지 라디안을 쓰는지 확인합니다.
    3. 결과 확인계산 결과와 최근에 계산한 식을 함께 확인합니다.
    4. 다시 계산괄호나 숫자를 바꿔 다른 경우도 바로 비교해 봅니다.

    계산할 때 알아둘 점

    기호를 이렇게 이해합니다.

    • ^는 제곱이나 거듭제곱을 뜻합니다.
    • log()는 상용로그, ln()은 자연로그입니다.
    • sqrt()는 제곱근입니다. 예를 들어 sqrt(16)은 4입니다.
    • 삼각함수는 선택한 각도 모드에 맞춰 계산됩니다.

    관련 팁

    실수를 줄이는 입력 팁

    • 괄호를 먼저 확인2+3*4와 (2+3)*4는 결과가 다릅니다.
    • 각도 단위 확인문제에서 도인지 라디안인지 먼저 확인한 뒤 계산하세요.
    • 너무 긴 식은 나누기중간값을 먼저 계산하면 입력 오류를 찾기 쉽습니다.

    계산 기준

    계산 전에 확인할 내용

    아래 내용은 계산 결과를 해석할 때 필요한 기준과 주의사항입니다.

    이 계산기는 무엇을 계산하나요?

    사칙연산, 괄호, 거듭제곱, 제곱근, 로그, 삼각함수를 입력한 순서와 연산 우선순위에 맞춰 계산합니다.

    입력값 설명

    • 계산식: 숫자, 연산자, 괄호, 함수명을 입력합니다.
    • 각도 모드: 삼각함수를 도(DEG) 또는 라디안(RAD) 기준으로 계산합니다.

    계산식

    입력식을 토큰으로 나눈 뒤 괄호, 함수, 거듭제곱, 곱셈·나눗셈, 덧셈·뺄셈 순서로 계산합니다.

    예시 계산

    DEG 모드에서 sin(30)+sqrt(16)을 입력하면 0.5+4로 계산해 4.5를 표시합니다.

    결과 해석

    결과가 예상과 다르면 괄호 위치와 DEG/RAD 모드를 먼저 확인하세요.

    주의사항

    보고서, 시험, 실무에 사용할 때는 유효숫자, 반올림, 단위 기준을 별도로 확인해야 합니다.

    참고한 공식 자료 또는 안내 문구

    수학 표기와 브라우저 JavaScript Math 함수 기준을 사용합니다.

    자주 묻는 질문

    공학용 계산기 자주 묻는 질문

    sin(30)은 0.5로 계산되나요?

    각도 모드가 도로 선택되어 있으면 sin(30)은 30도로 계산되어 0.5가 됩니다. 라디안 모드에서는 30라디안으로 계산됩니다.

    어떤 계산까지 할 수 있나요?

    현재는 sin, cos, tan, asin, acos, atan, log, ln, sqrt, abs, exp를 사용할 수 있습니다. 통계 계산이나 행렬 계산은 아직 제공하지 않습니다.

    계산 기록은 저장되나요?

    최근 계산 기록은 화면에서만 잠깐 보여줍니다. 새로고침하면 사라집니다.

    보고서나 과제에 그대로 써도 되나요?

    계산 확인용으로 쓰기 좋습니다. 보고서나 과제에 넣을 때는 유효숫자, 반올림 기준, 단위를 한 번 더 확인하세요.
