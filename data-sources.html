Data sources

데이터 출처 안내

한눈체크는 생활 속 확인이 필요한 정보를 정리하기 위해 공공데이터포털, 공식 공개 API, 브라우저에서 확인 가능한 기기 정보를 활용합니다. 사이트 내 결과는 각 제공기관의 데이터 반영 시점, API 응답 상태, 사용자의 입력값과 위치 기준에 따라 달라질 수 있습니다.

활용 데이터

  • 국세청 사업자등록정보 진위확인 및 상태조회 서비스 — 사업자등록 상태, 과세유형, 휴업·폐업 여부, 진위확인 결과 확인에 사용합니다.
  • 공정거래위원회 통신판매사업자 등록상세 제공 서비스 — 통신판매업 신고정보, 신고번호, 상호명, 대표자명, 사업장 소재지 등 공개 등록정보 확인에 사용합니다.
  • 한국환경공단 전기자동차 충전소 정보 — 충전소 위치, 충전기 타입, 충전기 상태, 운영기관, 주차료 무료 여부, 이용자 제한 여부 확인에 사용합니다.
  • 한국환경공단 에어코리아 대기오염정보 — 지역별 미세먼지, 초미세먼지, 오존, 통합대기환경지수 확인에 사용합니다.
  • 기상청 단기예보 조회서비스 — 외출 위험 종합 체크에서 강수확률, 강수형태, 기온, 습도, 풍속, 하늘상태 확인에 사용합니다.
  • 기상청 생활기상지수 조회서비스(3.0) — 외출 위험 종합 체크에서 자외선지수와 대기정체지수 확인에 사용합니다.
  • 카카오 Local API 및 Kakao Maps JavaScript API — 목적지 검색, 주소 좌표 변환, 전기차 충전소 지도 표시를 위해 사용합니다.
  • KAMIS 농산물유통정보 가격정보 Open API — 장보기 물가 확인 기능에서 농수축산물 가격정보를 표시하기 위해 사용합니다.
  • 한국소비자원 참가격 생필품 가격 정보 Open API — KAMIS에서 강한 매칭이 없는 가공식품·생활밀접 생필품 가격을 보조 확인하기 위해 사용합니다.
  • 국립중앙의료원 전국 응급의료기관 정보 조회 서비스 — 응급실 위치, 전화번호, 가용 병상, 중증질환 수용가능정보, 응급실·중증질환 메시지 확인에 사용합니다. 전국 응급실 기본정보 로컬 캐시는 기관명·주소·전화번호·좌표만 저장하고, 병상·중증질환 정보는 실시간 API 응답으로만 보강합니다. 야간 병원·야간 약국 로컬 캐시는 기관명·주소·전화번호·좌표·제공기관 운영시간만 저장하며 실제 진료·조제 가능 여부는 캐시하지 않습니다.
  • 국립중앙의료원 전국 병·의원 찾기 서비스 — 야간 병원 탭에서 병·의원명, 주소, 전화번호, 운영시간, 진료과목 참고에 사용합니다. 기본정보 로컬 캐시를 먼저 표시하고 API 응답으로 보강합니다.
  • 국립중앙의료원 전국 약국 정보 조회 서비스 — 야간 약국 탭에서 약국명, 주소, 전화번호, 운영시간 참고에 사용합니다. 기본정보 로컬 캐시를 먼저 표시하고 API 응답으로 보강합니다.
  • 지방행정 인허가 데이터 낚시터정보 — 낚시터 찾기 기능에서 낚시터명, 주소, 좌표, 전화번호, 주요어종, 이용요금, 안전시설, 편익시설 참고 정보 표시를 위해 사용합니다. 원본 CSV를 브라우저에서 직접 읽지 않고 지도 표시용 로컬 JSON 캐시로 정리해 사용합니다.
  • 지방행정 인허가 데이터 무료와이파이정보 — 무료 와이파이 찾기 기능에서 설치장소명, 주소, 좌표, 와이파이 이름, 시설구분, 제공기관, 관리기관 전화번호 참고 정보 표시를 위해 사용합니다. 원본 CSV를 브라우저에서 직접 읽지 않고 시도·시군구 단위 로컬 JSON 캐시로 정리해 사용합니다.
  • 지방행정 인허가 데이터 공중화장실정보 — 공중화장실 찾기 기능에서 화장실명, 주소, 개방시간, 장애인 화장실, 기저귀교환대, 비상벨, CCTV, 관리기관 전화번호 참고 정보 표시를 위해 사용합니다. 원본 CSV를 브라우저에서 직접 읽지 않고 주소 지오코딩 후 시도·시군구 단위 로컬 JSON 캐시로 정리해 사용합니다.
  • 공공데이터포털 — 위 API의 활용신청, 인증키 발급, 서비스 호출 경로를 제공합니다.

공공데이터 이용 표시

전기차 충전소 정보, 대기오염정보, 기상특보 등 공공데이터는 제공기관의 이용 조건에 따라 출처를 표시하고, 결과가 실제 현장 또는 최신 공식 안내와 다를 수 있음을 함께 안내합니다.

  • 전기차 충전소 정보: 한국환경공단 전기자동차 충전소 정보 OpenAPI 및 공공데이터포털 제공 정보 기준
  • 대기질 정보: 한국환경공단 에어코리아 대기오염정보 기준
  • 기상 정보: 기상청 단기예보 조회서비스와 생활기상지수 조회서비스 기준. 강수확률, 기온, 풍속, 자외선지수 등은 예보·지수 기준이며 실제 체감과 다를 수 있습니다.
  • 장소·좌표 검색: 카카오 Local API 및 Kakao Maps JavaScript API 기준. 의료기관 카카오맵 바로가기는 로컬 place_url 캐시가 있으면 바로가기를 사용하고, 없으면 카카오맵 검색으로 연결합니다. 전국 응급실 기본정보 로컬 캐시와 야간 병원·약국 기본정보 로컬 캐시는 기관명·주소·전화번호·좌표 같은 기본정보를 저장합니다. 병상·중증질환 수용 여부, 당일 진료·조제 가능 여부는 캐시하지 않습니다.
  • 장보기 물가 정보: KAMIS 농산물유통정보 가격정보 Open API 및 한국소비자원 참가격 생필품 가격 정보 Open API 기준
  • 응급실 정보: 국립중앙의료원 전국 응급의료기관 정보 조회 서비스 기준. 응급실 위치, 전화번호, 가용 병상, 중증질환 수용가능정보, 응급실·중증질환 메시지를 참고하며 실제 수용 가능 여부는 병원 상황에 따라 달라질 수 있습니다.
  • 낚시터 정보: 지방행정 인허가 데이터 낚시터정보 CSV 기준. 좌표가 유효한 항목만 지도용 로컬 캐시에 포함하며, 운영 여부·요금·예약 가능 여부·어종·시설 정보는 현장 사정에 따라 달라질 수 있습니다.
  • 무료 와이파이 정보: 지방행정 인허가 데이터 무료와이파이정보 CSV 기준. 좌표가 유효한 항목만 시군구 단위 로컬 캐시에 포함하며, 실제 접속 가능 여부·와이파이 이름 변경·시설 개방 여부는 현장 상황에 따라 달라질 수 있습니다.
  • 공중화장실 정보: 지방행정 인허가 데이터 공중화장실정보 CSV 기준. 주소를 카카오 Local API로 지오코딩한 좌표 캐시를 사용하며, 실제 개방 여부·청결 상태·시설 사용 가능 여부는 현장 상황에 따라 달라질 수 있습니다.

한눈체크는 원본 데이터를 그대로 보증하지 않으며, 사용자가 방문·결제·외출 전에 확인할 후보와 체크 포인트를 정리하는 참고 도구입니다.

신규 생활 확인 데이터 적용 상태

외출 위험, 응급실·야간 병원·약국 기능은 실제 API 연동 구조까지 반영되어 있습니다. 단, API 응답 상태, 기관별 데이터 갱신 시점, 현장 상황에 따라 결과는 달라질 수 있습니다.

  • 외출 위험 종합 체크 — 에어코리아 대기질, 기상청 단기예보, 생활기상지수를 합쳐 외출 위험도와 활동별 참고 문구를 생성합니다.
  • 응급실·야간 병원·약국 확인 — 국립중앙의료원 공공데이터 기준 응급실, 병·의원, 약국 후보를 표시하고 전화 확인을 우선 안내합니다. 카카오맵 장소 바로가기는 별도 로컬 캐시에 저장된 장소 링크만 사용하며, 병상·중증질환 수용 여부와 당일 야간 운영 상태는 캐시하지 않습니다.
  • 낚시터 찾기 — 지방행정 인허가 낚시터정보 CSV를 로컬 JSON 캐시로 정리해 지역별 낚시터 위치, 주요어종, 이용요금, 전화번호를 표시합니다. 실제 운영 여부와 요금은 방문 전 전화 확인이 필요합니다.
  • 무료 와이파이 찾기 — 지방행정 인허가 무료와이파이정보 CSV를 로컬 JSON 캐시로 정리해 지역·시군구별 무료 와이파이 위치, 와이파이 이름, 시설구분, 제공기관을 표시합니다. 실제 접속 가능 여부는 현장 확인이 필요합니다.
  • 공중화장실 찾기 — 지방행정 인허가 공중화장실정보 CSV를 주소 지오코딩 기반 로컬 JSON 캐시로 정리해 지역·시군구별 공중화장실 위치, 개방시간, 장애인 화장실, 기저귀교환대, 비상벨 여부를 표시합니다. 실제 개방 여부와 시설 상태는 현장 확인이 필요합니다.
  • 건강보험심사평가원 병원·약국 정보 — 향후 병원·약국 기본정보 보강이 필요할 때 보조 데이터로 사용할 수 있도록 환경변수 예시와 출처 안내에 반영했습니다.

공식 기관 사이트가 아닙니다

본 사이트는 국세청, 공정거래위원회, 공공데이터포털의 공식 서비스가 아니며, 각 기관과 제휴 또는 대행 관계에 있지 않습니다. 정확한 최종 확인이 필요한 경우 각 기관의 공식 조회 화면을 함께 확인해 주세요.

데이터 한계

조회 결과는 각 제공기관의 데이터 반영 시점, 신규 등록·변경 반영 지연, 사용자의 입력값, API 응답 상태에 따라 실제 상태와 차이가 있을 수 있습니다. 한눈체크는 데이터 표시를 돕는 도구이며 거래 결과, 충전 가능 여부, 외출 안전성, 건강상 판단을 보장하지 않습니다.

브라우저 기반 기기 정보

컴퓨터 사양 확인 기능은 별도 외부 데이터 제공기관을 사용하지 않고, 사용자의 브라우저에서 제공하는 Web API와 WebGL 정보를 바탕으로 운영체제 추정, CPU 논리 코어 수, 메모리 추정값, GPU/WebGL 정보, 화면 해상도 등을 표시합니다.

이 정보는 브라우저와 운영체제의 보안 정책에 따라 제공 범위가 제한될 수 있으며, 정확한 부품명은 사용자가 직접 확인해 입력해야 합니다.

주차비 확인 지도 안내

주차비 확인 지도는 공공데이터와 제공기관 데이터를 바탕으로 주차장 위치, 요금, 운영정보, 실시간 가능 대수 제공 여부를 참고용으로 표시합니다.

한눈체크

생활 속에서 확인이 필요한 정보를 한눈에 정리하고, 결과의 의미와 한계를 함께 설명하는 생활 확인 도구입니다. 조회·측정 결과는 참고용이며, 실제 판단을 보장하지 않습니다.
